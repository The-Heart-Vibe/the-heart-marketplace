# Heart Academic Spinouts Context

Loader kontekstu dla ventures wywodzących się z polskiego systemu akademickiego — tech transfer, IP, funding, cooperation patterns.

## Polski landscape akademicki dla VB

### Główni gracze (top tier research)

| Instytucja | Specjalizacja | Tech transfer office |
|------------|---------------|----------------------|
| **PAN** (24 instytuty) | Life sciences (IBB, IBD), Chemistry (IChF), Physics (IF), IT (IPI) | InnoTech PAN |
| **AGH** Kraków | Materials, energy, mining, AI | CTT AGH |
| **PW** (Warszawa) | Engineering, robotyka, AI, drones | CTT PW |
| **PWr** (Wrocław) | Electronics, photonics, semiconductors | Wrocławskie Centrum Transferu Technologii |
| **UW** | Computer science, mathematics, biology | UWRC (UW Research Center) |
| **UJ** Kraków | Life sciences, chemistry, JCET (cardiovascular) | CITTRU UJ |
| **WUM**, **GUMed**, **UM Łódź** | MedTech, clinical research | CTT medyczne |
| **UAM** Poznań | Chemistry, biology, optics | UCITT UAM |
| **Łukasiewicz** sieć (37 instytutów) | Applied research, industrial | Łukasiewicz CTT |

### Funding ścieżki dla research commercialization

| Program | Stage | Co finansuje |
|---------|-------|--------------|
| **NCN** (Narodowe Centrum Nauki) | Basic research | Publikacje, granty Sonata/Opus/Maestro — NIE commercialization |
| **NCBR** (Narodowe Centrum Badań i Rozwoju) | Applied research → commercialization | Szybka Ścieżka, LIDER, TANGO, Bridge Alpha — money flow to spinouts |
| **POIR / FENG** (UE funds) | Innovation → market | POIR 1.1.1 (badania), 2.1 (CBR-y firmowe), FENG 2.1 (komercjalizacja) |
| **ARP** (Agencja Rozwoju Przemysłu) | Spinouts / scaleups | Equity inwestycje, ARP Venture |
| **EU programs** | International scale | Horizon Europe (EIC Accelerator, Pathfinder), Eureka, Eureka Eurostars |
| **Bridge Alpha / Bridge Beta** (NCBR + VC) | Pre-seed / seed | Matching funds w spinout structure |

### Tax incentives dla R&D

- **Ulga B+R** (Tax Relief for R&D) — odpis 100% kosztów kwalifikowalnych, do 200% dla kosztów personalnych w CBR
- **IP Box** — 5% CIT na zyski z kwalifikowalnych IP (patenty, prawa autorskie, projekty wynalazcze) — kluczowe dla deep tech spinouts
- **CBR status** — Centrum Badawczo-Rozwojowe — dodatkowe ulgi + reputacja

## Typowe wyzwania research commercialization w PL

### 1. IP ownership — kto trzyma własność?

| Scenariusz | Domyślnie kto ma IP | Co zrobić |
|------------|---------------------|------------|
| Profesor zrobił badanie sam w trakcie obowiązków uczelnianych | **Uczelnia** (na podstawie umowy o pracę + regulaminów IP) | Negocjuj umowę licencyjną z CTT przed startem spinout |
| Postdoc/PhD na grancie NCN/NCBR | Uczelnia + agencja finansująca (NCBR ma prawa pierwokupu) | Sprawdź warunki grantu — niektóre wymagają commercialization plan |
| Badania w ramach Bridge Alpha | Spinout od początku (z partnerem VC) | Cleanest path, ale wymaga structured setup z NCBR |
| Wynalazek po godzinach (private) | Wynalazca | Rzadkie, wymaga proof że bez resources uczelni |

**Reguła Heart:** ZAWSZE wymagaj IP assignment / exclusive license PRZED ujawnieniem strategii commercialization. CTT może żądać % equity (typowo 5-15%) lub royalties.

### 2. Founder fit — profesor jako CEO?

**Anti-pattern:** profesor jako solo founder bez biznesowego doświadczenia.

**Typowe role splits w PL spinouts:**

| Rola | Kto pasuje | % equity (typowo) |
|------|-----------|---------------------|
| **Scientific co-founder / CSO / advisor** | Profesor (gardłowy/wizjoner) | 15-30% |
| **CEO / business co-founder** | Z zewnątrz (Heart często ją dostarcza) | 20-35% |
| **CTO** | Senior researcher z zespołu profesora (PhD, postdoc) | 10-25% |
| **Uczelnia (CTT)** | License equity | 5-15% |
| **The Heart / VC** | Capital + operating support | 15-30% |
| **Option pool** | Future hires | 10-15% |

**Dual employment red flag:** profesor zostaje w uczelni 100% i robi spinout "po godzinach" → konflikt interesów + IP problemy + slow execution. Lepiej: 50/50 lub sabbatical.

### 3. Pricing rynku akademickiego (gdy spinout sprzedaje BACK do uczelni/badawczych)

- Uczelnie publiczne kupują przez **przetargi** — długi cykl, niskie ceny, biurokratyczny proces
- Korporaty CBR — szybciej, ale wymagają proof of concept industrial scale
- International — academic licensing models (different pricing per region)

## Wzorce cooperation z academic foundera (model Heart)

### Wzorzec A: Heart jako external co-founder

```
Profesor X (CSO, 25%) + Heart-supplied CEO (30%) + Senior researcher (15% CTO)
+ Uczelnia (10% licencja) + Heart (20%) = clean cap table
```

### Wzorzec B: Heart jako venture builder z funded spinout

```
Heart funduje seed (€200-500k) i operating capacity przed VC seed round.
Spinout incubuje 6-12 mc w Heart, potem self-standing.
```

### Wzorzec C: Co-investment z uczelnianym CTT

```
Heart + ARP + CTT co-invest w spinout w exchange for licence.
Less equity dilution for founder, ale więcej stakeholders w cap table.
```

## Decision impact dla building spinout

| Aspekt | Reality dla PL academic spinouts |
|--------|-----------------------------------|
| **Time to MVP** | 12-24 mc (vs 3-6 dla typical SaaS) — głębsza tech wymaga walidacji |
| **Sales cycle B2B** | 6-18 mc do enterprise/research customers; 24+ mc do publicznych |
| **Talent pool** | PhD / postdoc available cheaply w PL (~€2-4k/mo gross) — fundamental advantage |
| **Competition** | Często global research grupy + 2-3 global startupy — moat = lokalna ekspertyza + szybkość |
| **Funding ladder** | Bridge Alpha (€200k) → NCBR Szybka Ścieżka (€2-5M) → EU EIC Accelerator (€2.5M) → VC seed |
| **Exit horizon** | 7-10 lat (deep tech longer than SaaS), strategic acquisitions często przez global majors |
