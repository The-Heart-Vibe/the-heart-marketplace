# Templates

This directory must contain the branded `.pptx` template files before the plugin can generate presentations.

## Required files

| File | Google Slides source |
|------|---------------------|
| `blank.pptx` | [The Heart blank presentation](https://docs.google.com/presentation/d/1WM2yqL2LUVq_tKyBwpwA5yVgUt3ioyCO_KNN_JNBH3g) |
| `pitchdeck-toolkit.pptx` | [Pitch Deck Toolkit](https://docs.google.com/presentation/d/1h0VwTbQNLZ-hXFEPY_cvJShJZVrtxpj4cYvVFnvuHXs) |

## How to add

Open each Google Slides link above → File → Download → Microsoft PowerPoint (.pptx) → place the file in this directory.

> **Note:** Templates are excluded from the repository because they are too large and contain proprietary brand assets. Each team member must download them manually.
